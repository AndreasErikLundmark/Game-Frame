## Overview

**Game-Frame** is a simple web template designed to manage game scores and high scores. It serves as a frontend demo that can be easily integrated with futuere game builds,

- **Score Display**: A customizable UI element to display the current score during gameplay.
- **High Score Tracking**: Backend API to store and retrieve high scores, ensuring players can compete for the top spot. Uses Java and Spring Boot for the backend, managing high scores and providing a RESTful API for game interaction.


1. run HighScoreApplication.java
2. open index.html
